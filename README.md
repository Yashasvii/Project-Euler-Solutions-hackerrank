This project contains two folders:
forProjectEuler: In this folder, problems are solved according to the site: https://projecteuler.net
The main focus here is to solve the given problem within 1 minute.
forHackerRank: In this folder, problems are solved according to the site: https://www.hackizerrank.com/contests/projecteuler/challenges
Here problems are more optimized to pass all Test Cases of the site.
